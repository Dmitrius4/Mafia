# Создадим README с инструкциями
readme = '''# 🎭 Бот Мафия для Telegram

Полнофункциональный бот для игры в Мафию в групповом чате Telegram.

## 📋 Требования

- Python 3.9+
- Библиотека `python-telegram-bot==20.7`

## 🚀 Установка

```bash
pip install python-telegram-bot==20.7
```

## 🔧 Настройка

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен и вставьте в файл `mafia_bot.py` в переменную `BOT_TOKEN`
3. Добавьте бота в групповой чат и дайте права администратора

## 🎮 Команды

### Основные (для всех)
| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |
| `/newgame` | Создать новую игру (только в группе) |
| `/join` | Присоединиться к игре |
| `/leave` | Выйти из игры |
| `/players` | Список игроков |
| `/roles` | Список всех ролей |
| `/rules` | Правила игры |
| `/vote @ник` | Проголосовать (во время голосования) |
| `/kill @ник` | Выстрелить (для Стрелка) |

### Ведущий (создатель игры)
| Команда | Описание |
|---------|----------|
| `/setroles роль1 роль2 ...` | Назначить роли игрокам |
| `/startnight` | Начать ночь |
| `/endnight` | Закончить ночь и показать результаты |
| `/startvote` | Начать дневное голосование |
| `/endvote` | Закончить голосование |
| `/cancel` | Отменить игру |

## 🎭 Роли

### 🟦 Мирные
- **Шериф** — проверяет роли, может убивать ночью, имеет полицейский участок
- **Сержант** — помощник шерифа, наследует роль при смерти шерифа
- **Доктор** — лечит игроков (2 раза за игру, не подряд)
- **Куртизанка** — забирает клиента, спасая от убийства
- **Журналист** — сравнивает 2 игроков на группировку
- **Бомж** — следит за игроком, видит убийцу
- **Почтальон** — проверяет одного, результат другому
- **Тюремщик** — заключает 2 игроков в тюрьму
- **Стрелок** — 3 пули, стреляет днём (не в 1-й день)
- **Амур** — выбирает 2 влюблённых
- **Судья** — решает: казнить или помиловать
- **Ветеран** — 3 защиты, убивает ночных посетителей

### 🟨 Нейтральные
- **Маньяк** — убивает ночью, побеждает в одиночку
- **Путана** — заражает чумой, побеждает когда все заражены
- **Ведьма** — контролирует действия других

### 🟥 Мафия
- **Босс Мафии** — +1 голос, видится мирным
- **Мафиози** — обычный член
- **Подручный** — помощник

### 🟫 Якудза
- **Босс Якудза** — +1 голос, ниндзя для шерифа
- **Якудза** — обычный член
- **Подручный** — помощник

## 📜 Правила

1. Игра делится на фазы: **Ночь** → **День** → **Голосование**
2. Ведущий управляет ходом игры
3. Все роли получают задания в ЛС ночью
4. Голосование открытое, против себя нельзя
5. При равенстве голосов решает Судья
6. У каждого игрока есть роль — нет обычных мирных!

## 🏆 Условия победы

- **Мирные** — нет мафии, якудза и маньяка
- **Мафия** — равенство или превосходство над остальными
- **Якудза** — равенство или превосходство над остальными
- **Маньяк** — остался один
- **Путана** — все заражены чумой

## ⚠️ Важно

- Бот должен иметь право отправлять ЛС игрокам
- Игроки должны начать диалог с ботом (/start) до игры
- В группе бот должен быть администратором

## 📄 Лицензия

'''

    f.write(readme)
